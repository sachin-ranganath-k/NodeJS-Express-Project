---------- Tools required ---------
    -- MongoDB
    -- MongoDB Compass

------- How to run after installation --------
    -- Copy path of MongoDB bin 
    -- Paste the path in Environment Variables (PATH section)
    -- Run mongod in command prompt
    -- Open MongoDB Compass and connect the connection with database


------- MongoDB teminologies ---------
Database -> Database
Collection -> Table
Documents -> Rows


---- Schema Types ----
Number
String
Boolean
Date
Buffer   ---> To store binary data
ObjectID ---> To assign unique Identifier
Array

npm i mongoose
--- Mongoose is a JavaScript object-oriented programming library 
    that creates a connection between MongoDB and the Node.js JavaScript 
    runtime environment.
